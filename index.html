AIGC / Full-stack / 3D Preview

AIGC 资产生成全栈平台

搭建支持图像、视频、3D 模型生成与在线预览的闭环平台，围绕异步任务编排、 Diffusers 服务化、资源管理和 Web 3D 展示，把 AI 资产从生成到交付串成完整流程。

  • 高并发异步队列承接耗时生成任务，降低前端等待和服务阻塞。
  • 将 Diffusers 推理能力服务化，加速图像与视频资产生成落地。
  • 支持 3D 模型在线预览，让生成结果可以即时检查与迭代。

《学园偶像大师》自动育成系统

实现集状态识别与策略优化于一体的游戏自动化系统，结合 OCR、模板匹配和强化学习策略， 对育成过程中的状态变化与决策收益进行自动化判断和迭代。

  • 用计算机视觉识别游戏画面状态，构建可执行的自动化控制链路。
  • 通过策略优化与强化学习思路持续改进行动选择。
  • 关注稳定性、可观测性和复杂流程中的容错恢复。
LLM / NLP / Hook / Learning Tool

unaTranslator 智能学习工具

二开定制集成大模型解析与游戏翻译的学习系统，把 LLM 语法解析、内存 Hook 文本注入、 NLP 处理和复习流程连接起来，让游戏文本从即时翻译延伸为可复盘的学习材料。

  • 接入大语言模型分析句法、词义和语境，提升翻译与学习解释质量。
  • 通过文本捕获与注入能力降低手动复制成本。
  • 将生词、例句和复习数据沉淀为闭环学习资产。
Wear OS / BLE / Time-series Data

Wear OS 上肢康复监测系统

开发软硬件协同的医疗康复数据采集系统，通过 Wear OS 与移动端双端架构、 低功耗蓝牙同步和毫秒级时序采样分析，为上肢康复训练提供更精准的监测数据。

  • 设计手表端与手机端协同的数据采集、同步和展示流程。
  • 使用 BLE 进行低功耗设备通信，适配康复训练场景。
  • 处理高频时序数据，为动作分析和康复评估提供基础。
